重要なHTMLタグ一覧

このページは、初心者シリーズWeb基礎編「HTML」で多用されるタグのいくつかをすぐ参照できるクイックリファレンスページとして作られます。このページの制作を通じて、HTMLのテーブル（table）をtableタグやそれに関するタグの使い方を学びます。

以下のテーブルに含まれるHTMLタグは全体の一部に過ぎませんが、重要なHTMLについてはここで参照できます。

ブロック要素

HTMLタグ タグ名 目的
h1–h6 見出し（heading） この見出しを置く（レベル： 1–6）
p 段落（パラグラフ：paragraph この中に文章を入れいる
table 表（テーブル：table この中にテーブルを置く
tr 表の行（table row） この中にデータを置く
th 表のデータ（table data) この中に表のセルデータを置く
header ヘッダー(header） 見出しを表す
div セクション（division） セクションを区切る
ol 順序付きリスト（ordered list 順序付きのリスト
ul 順序なしのリスト（unordered list） 順序なしのリスト
li リストアイテム（list item） リストアイテム

インライン要素

HTMLタグ タグ名 目的 記述例 画面上の表示結果
em 軽い強調（emphasized) テキストを軽く強調する 技術の成熟（<em>technical sophistication</em>） 技術の成熟（technical sophistication）
strong 強調（strong） テキストを強調する <strong>少なくとも10億人</strong> 少なくとも10億人
a アンカー(anchor） ハイパーリンクにする <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img 画像（image） 画像を配置する <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code コード（code） 文字幅を統一する <code>img</code> img
span スパン（span） インライン要素としてグループ化 <span>インライン要素としてグループ化</span> インライン要素としてグループ化
